插入排序

插入排序算法的原理可以举个例子来讲：想象一下你拿到一叠无序的纸牌，那么你从第二张纸牌开始给它排序的方式就是插入排序算法。 纸牌排序算法分析（从小到大）：首先从第2张牌开始，与前一张对比，如果前一张较大，那么把前一张向后放一个位置，把第2张插入第1张的位置。然后看第3张，与第2张比较，如果第2张大，那么第2张向后移动一个位置，然后比较第1张和第3张，如果第3张大，那么插入第2张的位置，否则，把第1张移到第2张位置，第3张最终第1个位置，以此类推即是插入排序。swift实现算法如下，

由小到大排序：

func insertionSortBigger(var array: Array<Int>) -> Array<Int>{
    for(var j = 1 ; j<array.count ; j++){//从第二个开始向前对比插入
        let key = array[j] //记录要比较的值
        var i = j-1
        while(i>=0 && array[i]>key){//如果key较小，那么现有的位置向后移，为key空出位置
            array[i+1] = array[i] //移位
            i--
        }
        array[i+1] = key
    }
    return array
}

由大到小排序：

func insertionSortSmaller(var array: Array<Int>) -> Array<Int>{
    for(var j = 1 ; j<array.count ; j++){//从第二个开始向前对比插入
        let key = array[j] //记录要比较的值
        var i = j-1
        while(i>=0 && array[i]<key){//如果key较大，那么现有的位置向后移，为key空出位置
            array[i+1] = array[i] //移位
            i--
        }
        array[i+1] = key
    }
    return array
}
